# 📊 HR Analytics Dashboard – Power BI

## 📌 Project Overview
Companies need to monitor and understand their workforce in order to improve talent management and decision-making.

This project analyzes an **HR dataset** containing employee information, salaries, departments, demographics, and employment history.  
The goal is to build an **interactive Power BI dashboard** that helps management explore workforce trends such as **headcount, retention, and turnover**.

---

## 🎯 Objectives
- Model HR data using a **Star Schema**
- Create important **HR KPIs** using **DAX**
- Build an **interactive multi-page dashboard**
- Provide clear insights about workforce structure and employee trends
- Deliver a **professional and portfolio-ready report**

---

## 🛠 Tools Used
- 📊 Power BI  
- 📐 DAX  
- 🔄 Power Query  
- 📁 CSV Data Sources  

---

## 🗂 Data Model
The dataset is structured using a **Star Schema**.

**Fact Table**
- `People Fact`

**Dimension Tables**
- Department
- Job Level
- Termination
- Demographics
- Education
- Location
- Marital Status
- Manager Hierarchy

Relationships and keys were created to ensure an optimized and scalable data model.

---

## 📈 Key KPIs
The dashboard includes several important HR indicators:

- 👥 **Total Employees**
- 📊 **Headcount**
- 🔄 **Retention Rate**
- 📉 **Turnover Rate**

These metrics are calculated using **custom DAX measures**.

---

## 📑 Dashboard Pages

### 👥 Headcount
- Total headcount KPI
- Employees by department and job level
- On-site vs remote employees
- Geographic distribution of employees
- Demographic insights (gender, race, age, education)

### 🔄 Retention
- Overall retention rate
- Retention by department and job level
- Starting vs ending headcount
- Interactive filters

### 📉 Turnover
- Overall turnover rate
- Turnover trend over time
- Termination type and reason
- Employee termination details
